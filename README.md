This is a [RUET TRANSPOST]('') project Create with [`React.js`](https://nextjs.org/).

## Getting Started

First, install all dependency using command added bellow:

```bash
npm install
# or
npm i
```
After installing all dependency, To run the code in locally just follow the command

```bash
npm run dev
```


Then it will Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
